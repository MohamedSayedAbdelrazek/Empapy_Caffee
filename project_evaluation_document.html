💎 المميزات الأساسية للمتجر

نظام إدارة المنتجات المتقدم

إضافة وتعديل وحذف المنتجات مع دعم الفئات المتعددة، الصور، الأوصاف، خيارات المنتج (الحجم، النوع، الإضافات) مع أسعار مختلفة لكل خيار، والحذف الناعم (Soft Delete) لاستعادة المنتجات المحذوفة.

نظام الفئات المرن

تنظيم المنتجات في فئات مختلفة مع صور مميزة لكل فئة، روابط ودية (Slugs)، وعرض عدد المنتجات في كل فئة تلقائياً.

سلة المشتريات الذكية (AJAX)

سلة تسوق تفاعلية تعمل بدون إعادة تحميل الصفحة، مع حفظ تلقائي للجلسة، تحديث الكميات فوري، وعرض ملخص الطلب بشكل ديناميكي.

نظام الدفع والطلبات

عملية دفع سلسة مع دعم الدفع عند الاستلام، حساب الشحن التلقائي، وتطبيق كوبونات الخصم مع التحقق الفوري.

نظام التقييمات والمراجعات

يسمح للعملاء بتقييم المنتجات وكتابة مراجعات، مع عرض متوسط التقييم وعدد المراجعات لكل منتج.

قائمة الأمنيات (Wishlist)

حفظ المنتجات المفضلة للمستخدمين المسجلين مع إمكانية الإضافة والإزالة بنقرة واحدة.

📱 تطبيق الويب التقدمي (PWA)

🚀 تجربة تطبيق أصلي على الويب

المنصة مبنية كتطبيق ويب تقدمي (Progressive Web App) يمكن تثبيته على الهاتف والحاسوب كتطبيق مستقل، مما يوفر تجربة سريعة وموثوقة حتى بدون اتصال بالإنترنت.

  • التشغيل بدون إنترنت (Offline Mode)
    صفحة offline مخصصة وجذابة مع تخزين مؤقت ذكي للأصول والصفحات المهمة باستخدام Service Worker.
  • قابلية التثبيت
    نافذة تثبيت احترافية مع دعم iOS وAndroid، أيقونات مختلفة الأحجام، وتجربة تثبيت سلسة.
  • التحميل السريع
    استراتيجيات تخزين مؤقت متقدمة (Cache-First للأصول، Network-First للمحتوى) لضمان سرعة التحميل.
  • اختصارات التطبيق (App Shortcuts)
    وصول سريع للمتجر وطلباتي مباشرة من أيقونة التطبيق على الشاشة الرئيسية.

🔔 نظام الإشعارات الفورية (Firebase Cloud Messaging)

إشعارات لحظية للإدارة والعملاء

نظام إشعارات متكامل باستخدام Firebase Cloud Messaging يدعم إشعارات الطلبات الجديدة، تغيير حالة الطلب، رسائل التواصل الجديدة، إلغاء الطلبات، والمستخدمين الجدد. يعمل في الخلفية وعند فتح التطبيق مع صوت مخصص وتأثيرات بصرية.

إشعارات ذكية حسب السياق

النظام يتعرف تلقائياً إذا كان التطبيق مفتوحاً ويمنع تكرار الإشعارات، مع عرض Toast داخل التطبيق وإشعار نظام عند الخلفية.

إدارة الأجهزة المسجلة

تسجيل وإلغاء تسجيل أجهزة المستخدمين تلقائياً، مع دعم أجهزة متعددة لكل مستخدم.

🎁 نظام الولاء والمكافآت المتكامل

نظام ولاء متقدم ومتعدد المستويات يشجع العملاء على التكرار والإحالة مع مكافآت قابلة للتخصيص.

🏆 مستويات العضوية (Tiers)

  • نظام مستويات متدرج: برونزي، فضي، ذهبي، بلاتيني مع مزايا متصاعدة لكل مستوى
  • ترقية تلقائية: ترقية المستوى تلقائياً عند تحقيق النقاط المطلوبة
  • مزايا حصرية: مضاعفات نقاط، خصومات إضافية، أولوية الدعم لكل مستوى

💰 قواعد كسب النقاط (Point Rules)

  • قواعد مخصصة: نقاط لكل عملية شراء، تسجيل، إحالة، طلب أول
  • مرونة كاملة: تحديد عدد النقاط لكل نوع حدث مع تمكين/تعطيل القواعد
  • شروط متقدمة: دعم قاعدة "الطلب الأول فقط" للترحيب بالعملاء الجدد

🎀 المكافآت (Rewards)

  • أنواع متعددة: خصم نسبي، خصم ثابت، منتج مجاني، شحن مجاني
  • إدارة المخزون: تحديد الكمية المتاحة لكل مكافأة مع تتبع الاسترداد
  • صلاحية محددة: تاريخ انتهاء للمكافآت المستردة

👥 نظام الإحالات (Referrals)

  • كود إحالة فريد: كود مخصص لكل مستخدم يُنشأ تلقائياً
  • مكافأة مزدوجة: نقاط للمُحيل والمُحال عليه
  • تتبع شامل: إحصائيات الإحالات الناجحة والمعلقة

⚙️ لوحة تحكم الإدارة المتقدمة

لوحة معلومات تحليلية

إحصائيات شاملة: إجمالي الإيرادات، عدد الطلبات، الطلبات المعلقة، المنتجات، العملاء، مع رسوم بيانية للأداء الشهري.

لوحة كانبان للطلبات

واجهة سحب وإفلات (Drag & Drop) لإدارة حالات الطلبات بأسلوب Kanban احترافي مع تأثيرات بصرية وصوتية.

إدارة الموظفين

أدوار متعددة (Admin, Cashier) مع صلاحيات مختلفة - الكاشير يرى الطلبات فقط، الأدمن لديه صلاحيات كاملة.

نظام الكوبونات

إنشاء كوبونات خصم بنسبة أو مبلغ ثابت، مع حد استخدام، تاريخ انتهاء، وحد أدنى للطلب.

إدارة رسائل التواصل

استلام وإدارة رسائل العملاء من صفحة "اتصل بنا" مع تحديث حالة الرسائل (جديد، مقروء، مُجاب).

إدارة العملاء

عرض قائمة العملاء مع تفاصيل الحساب، سجل الطلبات، ونقاط الولاء.

📈 قابلية التوسع والتطوير

هيكل كود نظيف

MVC Architecture مع فصل واضح للمسؤوليات، Controllers منظمة، Models مع علاقات Eloquent، Services للمنطق المعقد.

جاهز للـ Deployment

مُعد للنشر على Digital Ocean أو أي خادم Linux، مع ملفات التكوين الجاهزة.

قابل للتخصيص

يمكن تحويله لأي متجر آخر (ملابس، إلكترونيات، مطاعم) بتعديلات بسيطة.

وثائق ومراجعات

كود موثق مع تعليقات، Migrations واضحة، وسجل Git منظم.
